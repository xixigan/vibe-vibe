# 13.1 域名购买与 DNS 配置

> **待开发**

**预计涵盖内容：**

- 域名购买指南（注册商选择：阿里云、腾讯云、Cloudflare）
- 品牌策略：短、易记、易拼写；考虑续费成本
- 域名结构：根域名、顶级域名（.com/.cn/.io）、二级域名、子域名
- DNS 解析实战：
  - A 记录：指向 IP 地址（云服务器场景）
  - CNAME 记录：指向另一个域名（部署平台场景）
- 绑定域名到部署平台（平台自动配置 HTTPS 证书）
- 验证 HTTPS 是否生效

**本教程附带阿里云MCP配置指南，此后只需告诉 AI：**
> "配置 DNS 解析，将我的域名指向 EdgeOne Pages"
> "验证我的域名 HTTPS 是否配置正确"
