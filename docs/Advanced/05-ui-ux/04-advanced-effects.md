# 5.4 让页面更高级的效果

> **待开发**

---

有些页面一看就很"贵"，但你说不清为什么。其实它们用了很多标准化的高级效果。

本节会介绍：
- **滚动效果**：视差滚动、滚动触发动画、进度指示器
- **过渡效果**：3D 翻转、卡片堆叠、页面切换动画
- **加载体验**：骨架屏、懒加载图片、乐观更新
- **视觉特效**：毛玻璃、渐变边框、光晕效果
- **每个效果配一句提示词模板**

---

::: tip 少即是多

一个页面用 1-2 个高级效果就够了。全都要容易花里胡哨。

:::
