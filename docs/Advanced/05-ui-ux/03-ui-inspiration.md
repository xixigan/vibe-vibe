# 5.3 UI 风格与灵感

> **待开发**

---

你说"要一个现代风格的登录页"，AI 理解的"现代"和你想的可能不一样。用专业术语和参考网站，沟通会更高效。

本节会介绍：
- **灵感网站**：Awwwards、Dribbble、Mobbin、Land-book
- **风格关键词**：玻璃拟态、新粗野主义、极简主义、Bento 布局...
- **如何向 AI 描述风格**：用参考图 + 专业术语 + 具体场景
- **提示词模板**：把截图喂给 AI 后该怎么问

---

::: tip 别重复造轮子

看到一个喜欢的设计，先问 AI：这个效果叫什么？用什么库能实现？

:::
