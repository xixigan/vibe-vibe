# 5.2 动画与交互库

> **待开发**

---

一个恰到好处的动画，能让页面从"能用"变成"好用"。但过度动画会让用户头晕。

本节会介绍：
- Framer Motion —— React 生态最流行的动画库
- GSAP —— 专业级动画引擎
- Three.js —— 3D 效果的终极武器
- 轻量级 vs 重型效果的选择标准
- 一句话提示词模板

---

::: tip 动画的度

好动画应该是用户**几乎察觉不到**的——它只是让交互更流畅，而不是抢戏。

:::
