# 5.2 组件库

> **待开发**

---

组件库是预先做好的"乐高积木"。你不需要每次都手写按钮、输入框、对话框，直接拿现成的来用。

本节会介绍：
- **shadcn/ui** —— 轻量级、高度定制化，代码复制到你项目里
- **Ant Design** —— 企业级组件库，功能完整
- **Material-UI (MUI)** —— Google 设计语言，生态成熟
- **Radix UI** —— 无样式组件库，完全可定制
- **NextUI** —— 现代感强，默认好看
- 什么时候用哪个组件库
- 如何让 AI 用好组件库

---

::: tip 教程推荐 shadcn/ui

本教程推荐使用 **shadcn/ui**。它不是 npm 包，而是把代码直接复制到你的项目，你可以随意修改——非常适合 AI 辅助开发。

:::
