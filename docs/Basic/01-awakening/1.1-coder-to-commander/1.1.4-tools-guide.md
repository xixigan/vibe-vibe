# 1.1.4 这对你意味着什么

## 过去 vs 现在

让我们做一个直观的对比：

### 过去：想做一个小工具？

```
想法 → 学编程（3-6个月）→ 开始写代码 → 不断踩坑 → 
可能放弃 → 或者花几周做出来 → 勉强能用
```

**门槛高、周期长、容易放弃。**

### 现在：想做一个小工具？

```
想法 → 打开AI工具 → 说清楚你要什么 → 
几分钟后看到结果 → 不满意？继续说 → 当天就能用
```

**门槛低、周期短、立刻能用。**


## 具体能做什么？

你可能在想："说得好听，但我能做什么呢？"

来看看一些真实的例子：

### 给自己用

| 你的烦恼 | 你可以做 |
|---------|---------|
| 每天记不住喝了多少水 | 喝水打卡小工具 |
| 孩子的生日礼物清单总是乱 | 礼物管理小应用 |
| 想记录每天的开支 | 专属记账本 |
| 想背单词但市面上的App太复杂 | 简单的单词卡片 |

### 给工作用

| 你的烦恼 | 你可以做 |
|---------|---------|
| Excel做数据汇总太麻烦 | 自动化报表工具 |
| 客户信息散落在各处 | 简单的客户管理页面 |
| 团队排班总是冲突 | 排班协调小工具 |
| 会议纪要格式不统一 | 会议纪要模板生成器 |

### 给创业用

| 你的想法 | 你可以做 |
|---------|---------|
| 想验证一个产品概念 | 快速做出原型给用户看 |
| 想做一个小众工具卖钱 | 从MVP开始，边做边调整 |
| 想给自己的服务做个网站 | 几小时内上线一个落地页 |


## 真实案例：非开发者也能做出产品

这不是理论，来看几个真实的故事：

**案例1：产品设计师做出宠物App**

Cynthia Chen是一名产品设计师，从未写过代码。她用AI工具花了几周业余时间，做出了一款狗狗品种识别App"Dog-e-dex"，并成功上架App Store。

**案例2：旅游公司老板省下百万成本**

Shivek Sachdev在旅游行业工作，受不了Salesforce的高昂费用。他用AI工具自己做了一个定制CRM系统，满足了公司的特定需求（追踪航班预订、管理旅游套餐），省下了大量软件订阅费用。

**案例3：英语专业毕业生进入Notion实习**

WIRED杂志的记者（英语专业）去Notion公司体验Vibe Coding。她发现，这家估值100亿美元的公司，工程师们30-40%的代码都是AI辅助生成的。

这些不是特例。2025年的调研显示，**63%的Vibe Coding用户是非开发者**。


## 人人都能成为 Builder

"Builder"这个词很难翻译，它比"程序员"的含义更广。

一个 Builder 是**能把想法变成现实的人**。

以前，成为 Builder 需要：
- 会写代码，或者
- 有钱雇人写代码

现在，成为 Builder 只需要：
- 有想法
- 能把想法说清楚
- 愿意花时间和AI一起迭代

::: tip 核心观点
**Vibe Coding 不是让你成为程序员，而是让你成为 Builder。**

你的价值不在于"会写代码"，而在于"知道要做什么"和"能判断做得好不好"。
:::


## 你可能还有顾虑

### "我怕做不好"

别担心。Vibe Coding 的核心精神就是**"试错成本很低"**。

做得不好？改呗。
改崩了？让AI重来。
一个方案不行？换个思路再说。

你会在后面的章节学到，怎么保存进度、怎么回退版本，让你放心大胆地试。

### "我的想法会不会太简单"

恰恰相反。**简单的想法更容易成功**。

很多人失败是因为想法太复杂，而不是太简单。

后面的"心法"章节会详细讲这个——如何设计一个"刚刚好"的第一版。

### "AI做的东西能用吗"

能用。

对于个人工具、原型验证、小团队内部使用，AI生成的代码完全够用。

当然，如果你要做一个面向百万用户的产品，那就需要更专业的方案——但那是后话，也是进阶版教程会讲的内容。


## 也要知道边界

说了这么多好处，也要诚实地告诉你Vibe Coding的边界：

**适合用Vibe Coding的场景**：
- 个人工具、原型验证、内部使用
- 快速试错、验证想法
- 学习和探索

**需要更专业方案的场景**：
- 面向大量用户的生产系统
- 涉及敏感数据、金融交易的应用
- 需要长期维护和迭代的大型项目

Vibe Coding不是万能的。它是一个强大的起点，但如果你想走得更远，后面还有很多东西要学。

好消息是：**你不需要一开始就知道所有这些**。

先用Vibe Coding做出第一个东西，建立信心。剩下的，我们慢慢来。
